# Simple CRM Documentation

## Introduction

This is a simple Customer Relationship Management (CRM) application built with Python, Tkinter, and SQLite. It allows users to manage customer information through a graphical user interface.

## Features

*   Add new customers with name, email, and phone number.
*   View a list of all customers.
*   Update existing customer information.
*   Delete customers.
*   Simple and intuitive graphical user interface.

## Setup

1.  **Prerequisites:** Make sure you have Python 3 installed on your system.
2.  **Installation:** No installation is needed. Just make sure the following files are in the same directory:
    *   `main.py`
    *   `gui.py`
    *   `database.py`
3.  **Running the application:** Execute the `main.py` file to start the CRM application:
    ```bash
    python main.py
    ```
4.  **Database:** A `customers.db` file will be automatically created to store the customer data.
